# Ideas de copy

Documento vivo de lluvia de ideas de copywriting, separado del feedback de producto
(`docs/research/feedback/`). Acá se anotan frases sueltas por situación apenas se les ocurran —
en la calle, en la ducha, donde sea — sin filtrar. Las favoritas del momento se marcan
con ⭐ y suben al principio de su sección; las descartadas se dejan igual, no se borran,
por si sirven después.

## Cómo usar este documento

1. Cada situación del producto tiene su propia sección (crear una nueva si no existe).
2. Tirar ideas sin pulir — cantidad antes que calidad.
3. Marcar con ⭐ la favorita actual de la sección.
4. Si una frase tiene un problema conocido (tono, ambigüedad, etc.), anotarlo debajo en
   una sub-línea "Pendiente:" — no se descarta, se deja para pulir.

## Confirmación tras tomar un deseo (broma del dueño)

Reemplaza el "[Dueño] ya puede contar contigo" plano por algo que se sienta como si la
persona misma estuviera hablando — con humor, no solemnidad. Tiene que notarse
claramente que es una broma (riesgo: puede leerse como tristeza si no se pule el tono).

- ⭐ "Estás a punto de emocionarme."
  Pendiente: pulir tono — se puede confundir con tristeza en vez de alegría/broma.
- "No estoy llorando, es una pelusa en el ojo 🥹."
- "¿Llorando yo? TÚ estás llorando 🥹"
- "¿Llorando yo? Bah, estás loco. Me entró una pelusa." — versión implementada
  provisoriamente en el código (2026-07-19) mientras se pule la definitiva.
